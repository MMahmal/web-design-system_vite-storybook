# Web App Design System

## Goals
- Use vite as main build tool
- Build a design system for a web app using radix-ui
- Showcase components properties and behavior in Storybook

## How to run

### Init
```bash
npm install
```

### Run Dev Server
```bash
npm run dev
```

### Build App
```bash
npm run build
```

### Run Storybook
```bash
npm run storybook
```

### Build Storybook
```bash
npm run build-storybook
```
